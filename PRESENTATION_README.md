# Library Management System (Web)

This is a simple, responsive Library Management System web application.

- Add, view, search, update, and delete books
- Built with Node.js + Express (backend)
- Vanilla HTML/CSS/JS (frontend)
- Data persisted in `books.json`

Quick start

1. Install Node.js (if not already) or use the included portable Node inside `node_portable`.
2. From the project directory run:

```powershell
npm install
npm start
```

Open http://localhost:3000 in your browser.

This repository contains a minimal presentational README for demonstration purposes.
